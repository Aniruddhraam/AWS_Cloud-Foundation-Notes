## Vod 1 - Introduction to Cloud Computing

What is Cloud Computing?
On demand delivery of compute power with storage and with pay as you go option.
Infrastructure as hardware and self management of data center is more costlier

3 types of cloud service model  (More to less control over IT resources)
IaaS
Paas
SaaS

3 types of cloud deployment models:
Cloud
Hybrid
On-premises (Private cloud for enterprises)


## Vod 2 - advantages of the cloud
6 advantages of cloud Computing:
Trade Capital expense for variable expense
Massive economics for scale
Stop guessing capacity - Scale as you demand
Increase speed and availability
Stop spending money for running and maintaining data server 
Go Global in minutes
